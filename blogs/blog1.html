Pandas is a python library that specializes in handling data frames. In this article, we will look through how we can use pandas to conduct a descriptive analysis of a given data frame.

Describing the Data Frame:
Descriptive analysis consist of the following two statistics:

  1. Measures of central tendency.
  2. Measures of dispersion.
1. Measures of central tendency:
We can find measures of central tendency using the following methods:
  1. DataFrame.mean()- provides the mean of values across a given axis.
  2. DataFrame.median()- provides median across a given axis.
  3. DataFrame.mode()- provides mode of values across a given axis.

2. Measures of dispersion:
  1. DataFrame.min()- returns the minimum value across the given axis.
  2. DataFrame.max()- returns the maximum value across the given axis.
  3. DataFrame.var()- returns variance across the given axis.
  4. DataFrame.std()- returns standard deviation across the given axis.
Example:

The above mentioned examples can be used to describe specific columns of a data frame. If the number of columns in our dataset is huge, then we can use DataFrame.describe method to get a summary of all the columns at once.
